# GoDjangoDemo
#本项目使用Python Django框架来开发网页 实现一些小功能的demo        
功能点如下↓：</br>
1.理解Django的MTV模式(MVC)</br>
2.与服务器对话：理解HTTP协议</br>
3.用GET方法实现文章分类功能</br>
4.用POST方法实现Django表单</br>
5.用URL正则实现文章链接跳转</br>
等等…
</br>
